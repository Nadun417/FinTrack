◆ Core Features

Everything You Need to
Manage Your Money

FinTrack gives you powerful tools wrapped in a dark, elegant interface designed for clarity and speed.

Real-Time Charts

Doughnut, bar, and trend line charts update instantly as you log expenses. Spot patterns at a glance with interactive visualisations powered by Chart.js.

  • Spending by category (doughnut)
  • Category breakdown (bar chart)
  • Monthly spending trend (line)

Budget & Income Tracking

Set your monthly income and budget targets, then watch the progress bar fill up as expenses come in. See spent, remaining, and net savings at a glance.

  • Live budget progress bar
  • Colour-coded warnings
  • Net savings calculation

Custom Categories

Create unlimited color-coded categories to organise expenses exactly the way you think. Default categories are pre-loaded so you can start instantly.

  • Custom colour per category
  • Pre-loaded smart defaults
  • Filter & search by category

Multiple Profiles

Run separate budgets for personal expenses, family spending, side projects, or anything else. Switch between profiles in one click — each with its own categories, data, and currency.

  • Unlimited profiles per account
  • Independent categories each
  • 11 currency options

Private & Secure

Your data is stored in a Supabase PostgreSQL database with Row Level Security — each user's data is isolated so no one else can access it.

  • Row Level Security (RLS)
  • HTTPS encrypted connection
  • Per-user data isolation

Export & Backup

Download the current month as a CSV spreadsheet, or export your entire account as JSON for full backup. Restore everything with one-click import — never lose data.

  • Monthly CSV export
  • Full JSON backup & restore
  • Safe CSV injection protection

◆ FAQ

Frequently Asked Questions

Is FinTrack really free?

Yes, 100% free. There are no premium plans, no ads, and no hidden fees. All features are available to every user.

Where is my data stored?

Your data is stored in a secure Supabase PostgreSQL database with Row Level Security. Each user's data is completely isolated — no one else can access it.

Can I use it on mobile?

Yes! FinTrack is fully responsive and works in any modern browser on phones, tablets, and desktops.

How do I export my data?

Click "Export" in the header for a CSV of the current month, or go to Profile Settings for a full JSON backup of all data. You can re-import the JSON backup at any time.

What currencies are supported?

FinTrack supports USD, EUR, GBP, JPY/CNY, INR, KRW, AUD, CAD, CHF, BRL, and LKR. You can set a different currency per profile.

Can I delete my account?

You can reset all profile data or delete your entire account from the Profile Settings modal.
